Strona stworzona na potrzeby projektu stażu technicznego informatyka.

Projekt składa sie z strony internetowej oraz prezentacji wyswietlanej na podstronie
Strona internetowa zawiera przekierowanie do innnych stron
Przy pisaniu strony użyłem HTML, CSS oraz Java Script

Strona Główna jest to początkowa strona, na którą każdy użytkownik trafia
Nav-Bar czyli inaczej pasek nawigacji dzięki któremu możemy trafiac na inne strony oraz podstrony
Pozwala nam go rozwijac przy skalowaniu Java Script
Podstrona o nazwie "Prezentacja" wyświela animacyjnie prezentacje na temat projektu
Ostatnia podstrona o nazwie "Opis" jak sama nazwa wskazuje opisuje projekt który przygotowałem
